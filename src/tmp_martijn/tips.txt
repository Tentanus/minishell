hey Mares,
Dit is een document waar ik tips voor je tijdlijk opschrijf zodat ik ze niet
vergeet. 

 --	Probeer je code met zo min mogelijk comments te vullen.
	het zorg ervoor dat je effectief minder van je werkende code ziet en dit
	zorgt ervoor dat je minder snel door je code leest en vervolgens minder
	snel schrijft.

 --	Als je comments schrijf voor een probleem dat je later nog opgaat lossen
	los dat op een gegeven moment ook op. Er staan comment over segfaults die 
	3 maanden terug erin zijn gezet. 
	Ik heb zelf als regel dat ik als ik een fout vind ik het direct oplos.

 --	Probeer tijdens het schrijven van Code alvast norm te schrijven.
	Het is even lastig maar na een paar weken denk je er nauwelijks meer aan.
